Featured Projects

IDEal Classroom

Created as group project for EGR302: Engineering Design and Documentation for Dr. Mikyung Han (Client) and Dr. Kyungsoo Im (Instructor/Advisor)

IDEal Classroom is an online application that allows professors to create coding homework assignments for use in software and computer science classes.
Professors can easily upload custom grading scripts to each assignment so that the assignments are automatically graded after the deadline.
Students can create accounts and register for certain classes which allow them to access classes and upload assignments as well as get instant feedback once the assignment is graded.

Languages: Python (with Django), HTML, CSS (with Bootstrap), Javascript, Java

Barcode Generation API

A Flask-based barcode generation REST API that allows for the creation of many types of barcodes (including Code128, Code39, EAN8, and more) as well as QR Codes.

This service allows you to make calls to a web server and pass in the type of barcode you want to have generated and the data as well as optional parameters for the size and margins of the barcode, which is then returned to you.

Languages: Python (with Flask)
