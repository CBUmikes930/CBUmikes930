Featured Projects

Grand Prix Race Controller

A NodeJS-based web app that manages racers in a double-elimination tournament. The program allows for the registration of racers, the running of races and tracks the status of racers as they win and lose races until they are either eliminated or make it to the finals. The final race ends with a 1st, 2nd and 3rd place winner.

This system was created using a NodeJS server to control and track the race progress, a MongoDB database to hold the racers data and race history, and Sockets to update connections to display the current racers as well as the racers that are up next.

Languages: Javascript (NodeJS, JQuery, ReactJS), HTML, CSS, MongoDB

IDEal Classroom

Created as a group project for EGR302: Engineering Design and Documentation for Dr. Mikyung Han (Client) and Dr. Kyungsoo Im (Instructor/Advisor)

IDEal Classroom is an online application that allows professors to create coding homework assignments for use in software and computer science classes.
Professors can easily upload custom grading scripts to each assignment so that the assignments are automatically graded after the deadline.
Students can create accounts and register for certain classes which allow them to access classes and upload assignments as well as get instant feedback once the assignment is graded.

Languages: Python (Django), HTML, CSS (with Bootstrap), Javascript, Java

Barcode Generation API

A Flask-based barcode generation REST API that allows for the creation of many types of barcodes (including Code128, Code39, EAN8, and more) as well as QR Codes.

This service allows you to make calls to a web server and pass in the type of barcode you want to have generated and the data as well as optional parameters for the size and margins of the barcode, which is then returned to you in the form of a PNG image.

Languages: Python (Flask)
